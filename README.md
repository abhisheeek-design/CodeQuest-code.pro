# CodeQuest Deliverables

This folder contains two generated starting points:

- `codequest-web/`: Static web prototype. Open `index.html` in a browser.
- `codequest-flutter/`: Flutter app structure with a pixel-themed `LevelSelectScreen`.

The web prototype includes:

- HTML, Java, and Python learning tracks
- Beginner, intermediate, and advanced level states
- XP, streak, badges, leaderboard, and level-select map styling
- Live HTML preview editor
- Quiz feedback with retro success and try-again modal states
- Dark/light mode toggle

The Flutter scaffold includes the requested `MaterialApp` theme, `Color(0xFF0D0D0D)` background, pixel font family configuration, and track cards with `START` buttons.
